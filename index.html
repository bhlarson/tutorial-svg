SVG Elements in HTML

SVG is an XML-based markup language for describing two-dimensional based vector graphics. It can be embedded in HTML, generated or manipulated in Javascript, or in libraries like D3JS

SVG Introduction:

  
    <svg width="100" height="100">
      <circle cx="50" cy="50" r="40" stroke="red" stroke-width="4" fill="yellow" />
    </svg>
  • An SVG image begins with an <svg> element
  • The width and height attributes of the <svg> element define the width and height of the SVG image
  • The <circle> element is used to draw a circle
  • The cx and cy attributes define the x and y coordinates of the center of the circle within the SVG image. If cx and cy are not set, the circle's center is set to (0, 0)
  • The r attribute defines the radius of the circle
  • The stroke and stroke-width attributes control how the outline of a shape appears. We set the outline of the circle to a 4px green "border"
  • The fill attribute refers to the color inside the circle. We set the fill color to yellow
  • The closing </svg> tag closes the SVG image

SVG Lines

SVG <line> defines a line segment between two points.

  
    <line x1="5" y1="5" stroke="red" x2="90" y2="90" />

Line properties include:

  • stroke-width
  • stroke: specificing line color
  • stroke-linecap: can have vlues butt, round, and square

SVG Rectangle

SVG <rect> defines a rectangle by top-left poxition, width, and height

  
   <rect x="25" y="25" height="50" width="70"/>

Rectangle properties include:

  • stroke-width: stroke defining line color, stroke-linecap,
  • stroke: line color
  • fill
  • rx and ry corner radius
